Display

El display es la propiedad que define el tipo de caja que va a tener un elemento.

Valor Descripción
block El elemento se muestra como un bloque, ocupando todo el ancho disponible.
inline El elemento se muestra como una línea, ocupando solo el ancho que necesita.
inline-block El elemento se muestra como una línea, pero permite definirle un ancho y alto.
none El elemento no se muestra.

Position

La propiedad position define el tipo de posicionamiento que va a tener un elemento.

Valor Descripción
static El elemento se posiciona de forma estática, es decir, se muestra en el orden en que aparece en el código HTML.
relative El elemento se posiciona de forma relativa, es decir, se muestra en el orden en que aparece en el código HTML, pero permite moverlo con las propiedades top, bottom, left y right.
absolute El elemento se posiciona de forma absoluta, es decir, se muestra en el orden en que aparece en el código HTML, pero permite moverlo con las propiedades top, bottom, left y right, y se posiciona respecto al primer elemento padre que tenga un position diferente a static.
fixed El elemento se posiciona de forma fija, es decir, se muestra en el orden en que aparece en el código HTML, pero permite moverlo con las propiedades top, bottom, left y right, y se posiciona respecto a la ventana del navegador.
